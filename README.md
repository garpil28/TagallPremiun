Bot Telegram untuk sistem premium auto tagall multi-user.

- Sistem premium 30 hari (/addprem)
- Menu interaktif
- Limit tagall 1× per hari
- Backup otomatis harian
- Logs harian
- Notifikasi masa aktif ke owner
- Restart otomatis 00:00
